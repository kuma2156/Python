## 🧠 기억할 개념들 요약

### ✅ Python 기초 문법

- 변수 선언: `x = 10`, `name = "명규"`
- 출력: `print("Hello, world!")`
- 연산자: `+`, `-`, `*`, `/`, `//`, `%`, `**`

### ✅ 조건문 & 반복문

- 조건문: `if`, `elif`, `else`
- 반복문: `for`, `while`
- 반복 제어: `break`, `continue`

### ✅ 함수(Function)

- 정의: `def 함수명(매개변수):`
- 반환: `return` 문
- 기본값 매개변수, 키워드 인자 등 활용

### ✅ 주요 자료형

- 리스트: `[]` / `.append()`, `.remove()`, `.sort()`
- 튜플: `()` / 불변성
- 집합: `{}` / 중복 제거, 집합 연산
- 딕셔너리: `{key: value}` / `.keys()`, `.values()`
- 문자열: 슬라이싱 `[start:end]`, `.split()`, `.replace()`, `.find()`

### ✅ Pandas

- `import pandas as pd`
- `DataFrame`, `Series` 생성
- `.head()`, `.tail()`, `.info()`, `.describe()` 등 데이터 요약
- `.loc[]`, `.iloc[]`로 행/열 접근
